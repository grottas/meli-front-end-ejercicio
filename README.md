## Dependencias

- [Nodejs](https://nodejs.org/es/) > v6.0
- [NPM](https://www.npmjs.com/)

## Setup

Instalar dependencias del proyecto:

cd app
npm install
DEBUG=app:* npm start
DEBUG=app:* npm run devstart
